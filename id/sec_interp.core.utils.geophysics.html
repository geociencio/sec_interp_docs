sec_interp.core.utils.geophysics.calculate_geometric_factor(array_type: ArrayType, params: dict[str, float]) → float

Calculate the geometric factor (K) for various electrode arrays.

Parameter:

  • array_type -- The type of the array.

  • params -- Dictionary with required parameters: - Wenner: 'a' - Schlumberger: 'ab_2' (half current spacing), 'mn_2' (half potential spacing) - Dipole-Dipole: 'a' (dipole length), 'n' (separation multiplier)

Kembali:

Geometric factor K.

Return type:

float
